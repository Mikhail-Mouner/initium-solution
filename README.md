### Installation

```sh
git clone https://github.com/Mikhail-Mouner/initium-solution.git
```
```sh
cd initium-solution
```

```sh
cp .env.example .env
```

```sh
composer install
```

```sh
npm install && npm run dev
```

```sh
php artisan key:generate
```

Create Table Name( initium )

```sh
php artisan migrate --seed
```

```sh
php artisan serve
```
